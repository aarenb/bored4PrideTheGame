# Bored4Pride: The Game

_**Bored4Pride: The Game**_ is a 2D RPG game about the twitch charity event Bored4Pride. Bored4Pride is a yearly event, that takes place in June to raise money for a lgbtq+ charity. It's run by a group of streamers called the Bored Chaps, which is led by the streamer LordWongy. In this game you play as a gay star, which has to help save the charity event from a follow bot invasion.
